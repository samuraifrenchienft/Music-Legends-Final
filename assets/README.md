# Music Legends Assets

## Folder Structure

```
/assets/
  /frames/
    - community_frame.png (600x900px)
    - gold_frame.png (600x900px)
    - platinum_frame.png (600x900px)
    - legendary_frame.png (600x900px)
  /badges/
    - community_badge.png (80x80px)
    - gold_badge.png (80x80px)
    - platinum_badge.png (80x80px)
    - legendary_badge.png (80x80px)
  /logo.png (transparent background)
```

## Instructions

1. Place your 4 card frames in `/frames/` folder
2. Place your 4 tier badges in `/badges/` folder
3. Place your Music Legends logo as `/logo.png`

All images should be PNG format with transparency where needed.
